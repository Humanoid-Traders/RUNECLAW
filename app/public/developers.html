ERC-8257 Agent Tool Registry

RUNECLAW's read-only intelligence is packaged as an on-chain-discoverable tool (ERC-8257, the permissionless Agent Tool Registry that layers on the same ERC-8004 identity standard our track record anchors to on Base). Free and open: no pricing, no access gate — and structurally read-only.

GET /.well-known/ai-tool/runeclaw-intel.jsonthe canonical tool manifest (keccak256-hashed for the registry)

The verification ethos

Don't trust the dashboard — verify the fills. The Proof-of-PnL bundle canonicalizes as JSON with recursively sorted keys, no whitespace, UTF-8 (every number is already a string); publish_hash = SHA-256(canonical). Your agent can re-derive it in any language and reject anything that doesn't match. Safety reads are heuristic flags with reasons, never verdicts. Past performance never predicts future results.

Trade-capable MCP tools are a separate, operator-gated decision and are not exposed here.
