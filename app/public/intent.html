Intent Compiler

A wish is not a policy. “Only majors, max 5% per trade, no shorts, stop if down 8%” is what you mean — but an agent can only be trusted with money if that sentence becomes something a machine can check before it signs. The Intent Compiler turns plain words into a deterministic, revocable Authority Envelope: typed rules, each tagged with who enforces it — the wallet, the risk gate, or a human approval. This is the Guardian principle in one screen: the AI proposes, deterministic controls authorize, the wallet enforces.

Authority Envelope — compiled rules

This is a compiler preview: it turns language into a policy so you can read exactly what would be enforced. It binds nothing, signs nothing and moves no funds — in the app, a compiled envelope is tighten-only (it can only narrow the engine's caps), previews before it binds, and is revocable at any time. It is a public demo, so it never shows a dollar figure; a dollar-denominated approval limit compiles to an approval rule whose exact number you set privately in the app. A safety aid, not investment advice.
